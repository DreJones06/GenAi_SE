# ✨ Resurrection of Christ - Prompt Guide

This document contains both a **free-form prompt** and a **structured JSON prompt** for generating a cinematic AI image of the *Resurrection of Christ (Easter)*.

---

## 📝 Free Prompt

```
Resurrection of Christ at an ancient Jerusalem tomb at dawn, cinematic and hyper-realistic, epic spiritual atmosphere.
```

### 🔍 Explanation

* A **minimal prompt** designed for quick generation.
* Works well with tools like Midjourney or DALL·E when you want fast results.
* Focuses on:

  * Subject: Resurrection of Christ
  * Setting: Ancient tomb
  * Style: Cinematic, hyper-realistic

### 🔍 Explanation

The JSON prompt provides **fine-grained control** over image generation:

* **Style** → Defines overall visual tone (cinematic, epic, spiritual)
* **Scene** → Sets environment, time, and mood
* **Subject** → Detailed description of Jesus Christ
* **Composition** → Camera angle, framing, and depth
* **Lighting** → Critical for cinematic realism (god rays, rim light)
* **Effects** → Adds atmosphere and symbolic storytelling
* **Color Palette** → Controls emotional tone through colors
* **Quality** → Ensures high-resolution, realistic output
* **Optional Elements** → Enhances storytelling (angels, guards, doves)
