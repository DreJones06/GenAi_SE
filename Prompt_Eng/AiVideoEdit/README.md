
# 🎥 Cinematic Car Chase Prompt (Seedance 1.5)

## 📌 Project Description
This repository contains a **high-quality JSON prompt** designed for generating a cinematic car chase video using **AI text-to-video models** like Seedance 1.5.

The goal is to achieve a **Hollywood-style action sequence** with realistic physics, dynamic camera work, and dramatic lighting.

---

## 🧠 Prompt Breakdown

### 🎬 1. Main Prompt
Defines the **core scene and storytelling elements**:
- Neon-lit futuristic city
- High-speed sports cars
- Police chase dynamics
- Action intensity and cinematic tone

👉 This is the most important part—it drives the overall output.

---

### 🚫 2. Negative Prompt
Controls **what to avoid**:
- Low quality visuals
- Unrealistic physics
- Artifacts, glitches, or distortions
- Cartoon/anime styles

👉 Helps maintain realism and cinematic quality.

---

### 🎨 3. Style
Defines the **visual identity**:
- Cinematic
- Realistic
- Action-packed
- Blockbuster feel

---

### 🎥 4. Camera Settings
Controls how the scene is visually captured:

#### Shots:
- Drone shots (wide cinematic feel)
- Tracking shots (speed and motion)
- Close-ups (detail and intensity)

#### Movement:
- Fast-paced
- Dynamic
- Smooth but aggressive

#### Effects:
- Motion blur → enhances speed
- Lens flare → cinematic realism
- Depth of field → focus control
- Slow motion → dramatic highlights

---

### 💡 5. Lighting
Defines the **mood and atmosphere**:
- Neon glow
- Police siren lighting
- High contrast shadows
- Wet reflections

👉 Critical for cinematic realism.

---

### 🌆 6. Environment
Describes the **world setting**:
- Futuristic cityscape
- Skyscrapers
- Busy streets
- Fog + neon ambience

---

### ⚡ 7. Motion
Controls **action dynamics**:
- Extreme speed
- Drifting and sharp turns
- Near collisions
- Sparks and debris

---

### ⚙️ 8. Render Settings
Technical output configuration:
- 4K resolution
- 24 FPS (cinematic standard)
- 5–10 seconds duration

---

## 🚀 How to Use

1. Go to **OpenArt → Create Video**
2. Select **Seedance 1.5**
3. Paste the JSON from `prompt.md`
4. Generate video
5. Iterate by tweaking:
   - Duration
   - Camera shots
   - Motion intensity

---

## 🔥 Pro Tips

- Increase duration to **15–20 seconds** for better storytelling
- Add specific car brands (e.g., Ferrari, Lamborghini) for realism
- Introduce weather (rain, smoke) for dramatic effect
- Use multiple generations and pick the best result
