Playwright is a powerful Python library for browser automation and end-to-end testing, supporting Chromium, Firefox, and WebKit.

Installation

Install via pip and set up browsers with these commands:
github
+1

text
pip install playwright pytest-playwright
playwright install
Basic Setup

Use the context manager for sync API to launch browsers and pages:
blog.apify

text
from playwright.sync_api import sync_playwright, expect

with sync_playwright() as p:
    browser = p.chromium.launch(headless=False)
    page = browser.new_page()
    page.goto("https://example.com")
    browser.close()

Assertions

Use expect for reliable checks:
linkedin

text
expect(page).to_have_title("Dashboard")
expect(page.locator("h1")).to_have_text("Welcome")
expect(page).to_have_url("/dashboard")
expect(page.locator("button")).to_be_visible()
