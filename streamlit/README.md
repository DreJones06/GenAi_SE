# 🤖 CodeBuddy – AI-Powered Code Learning Platform

CodeBuddy is an interactive AI-driven code learning platform built using Streamlit, designed to help beginners write, execute, and understand code with real-time intelligent feedback.

Powered by Groq’s Llama 3.3 70B model, CodeBuddy enables users to learn programming faster through AI-assisted explanations, corrections, and insights.

---

## 🚀 Features

- 🧠 **AI-Powered Code Review**
  - Get instant feedback on your code
  - Suggestions for improvement and optimization

- 💻 **Interactive Code Execution**
  - Write and test code in real-time

- 📚 **Beginner-Friendly Learning**
  - Simplified explanations for complex logic
  - Step-by-step guidance

- ⚡ **High-Speed AI Inference**
  - Integrated with Groq’s ultra-fast LLM backend

- 🌐 **Web-Based UI**
  - Built using Streamlit for quick and clean interface


---

## ⚙️ Tech Stack

- **Frontend/UI:** Streamlit  
- **Backend Logic:** Python  
- **Automation Engine:** Playwright  
- **AI Model:** Groq Llama 3.3 70B  
- **Async Processing:** asyncio
